# Concept Explanation Prompt Template

Use this prompt whenever you want a concept explained in **table format** and saved as a `.md` file.

---

## Prompt

```
Explain [CONCEPT_NAME] in table form with real-world examples.
Save the result in understand_code_IQ folder.
```

### Example Usage

```
Explain "Hoisting in JavaScript" in table form with real-world examples.
Save the result in understand_code_IQ folder.
```

```
Explain "async/await vs Promises" in table form with real-world examples.
Save the result in understand_code_IQ folder.
```

```
Explain "var vs let vs const" in table form with real-world examples.
Save the result in understand_code_IQ folder.
```

---

## What you'll get

Each saved `.md` file will contain:

| Section | Description |
|---------|-------------|
| **Title** | Concept name with `#` heading |
| **Comparison Table** | Key differences side-by-side |
| **Examples** | Code snippets for each variant |
| **Visual Flow / Diagram** | ASCII or mermaid diagram where applicable |
| **Key Takeaways** | Bullet-point summary |

---

## Folder

All files are saved to: `D:\Playwrite_3X\understand_code_IQ\`
